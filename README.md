# Fluentian Internship  Authentication API

Built by Rebika

## How to run
1. Activate venv: `venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Mac/Linux)
2. Install packages: `pip install -r requirements.txt`
3. Run the server: `uvicorn main:app --reload`
4. Open browser: http://127.0.0.1:8000/docs

## Features I implemented
 Register, Login, /me, Delete account
Email verification (send-code + verify)
 JWT authentication
 SQLite database
 Hashed passwords
 Proper error messages
